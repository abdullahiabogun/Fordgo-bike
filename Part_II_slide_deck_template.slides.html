Part II - (Interactions between duration and features of trips taken.)¶

by (Abdullahi Abogun)¶

Investigation Overview¶

In this investigation, i set out to see which features mostly interacted with the duration of trips. The primary focus were the time(from which i was able to extract day of week and time of day),user type,birth year(from which i was able to create an age column) and station.

Dataset Overview¶

The data set includes information about individual rides made in a bike-sharing system covering the greater San Francisco Bay area. It contains anonymized data of users for an approximate 183,000 trips, inluding duration and other attributes. Missing age were replaced with the mean age and no row or column was dropped.

(Visualization 1)¶

The duration distribution takes up values between 1 and 1,400. When plotted on a log scale we see two(2) breaks, first around 1.5 and 2 and another between 3 and 4. We the see two(2) peaks with the first between 4 and 5 minutes and the second between 15 and 20 minutes which depicts a bimodal distribution.
